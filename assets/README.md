# Assets Directory

This directory should contain:

## Images Needed:
1. **phone-mockup.png** - Mobile app mockup for hero section
2. **logo.png** - Aegis Travel logo
3. **hero-bg.jpg** - Background image for hero section (optional)

## You can use:
- Free mockup generators like Mockuphone, Smartmockups
- Icon generators for logo
- Stock photo sites like Unsplash, Pexels

## Recommended Sizes:
- phone-mockup.png: 350x650px (transparent background)
- logo.png: 512x512px (transparent background)
- hero-bg.jpg: 1920x1080px

The website works perfectly without these images as fallbacks are in place!
